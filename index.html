Coffee Lover Project

Welcome to Bootstrap project: Coffee Lover

You will recreate this webpage and an additional Form Page

Just follow the instruction steps here to re-create these webpages!

1. Create a general landing page with a jumbotron
2. Next create a NavBar on the Page that links to another html file
3. Add two paragraph of lorem below jumbotron
4. Next use gride system to add thumbnail pictures of coffee. You will need to use thumbnailclass for this.

Here are the image links:

One Two Three Four Five Six

5. Next make sure that the largest setting you have 3 columns of picture. On the smallest screen setting you should hane two columns. Its upto to you where you want to change.
6. Next create another html file for sign up page.
7. Create a form on this Sign Up page with Email, Password and Check Box


That's it. Feel Free to around this project and style it more to your liking. The main focuses of this project are to use NavBar, Jumbtron, Fotm, Container and the Grid System.
